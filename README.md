# Book Recommendation System using KNN (Python & scikit-learn)

**Overview:**  
Machine learning project that builds a book recommendation system using K-Nearest Neighbors (KNN). Analyzes user-book rating data to recommend similar books based on distance metrics.

**Technologies Used:**  
- Python 3.x  
- Pandas & NumPy  
- scikit-learn  
- Matplotlib (optional visualization)  

**Features:**  
- Data cleaning and preprocessing of large rating dataset  
- Filtering users and books for statistical significance  
- KNN-based similarity model using NearestNeighbors  
- Recommends similar books based on input title  

**Project Workflow:**  
1. Load and preprocess dataset  
2. Filter users and books based on rating thresholds  
3. Create user-item matrix  
4. Train KNN model using similarity metrics  
5. Generate recommendations for a given book  

**Key Skills Demonstrated:**  
- Recommender systems and collaborative filtering  
- Data preprocessing and feature engineering  
- KNN model implementation using scikit-learn  
- Handling large-scale datasets
